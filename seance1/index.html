Trois composantes fondamentales différencient le Web de l'Internet

  1. Un langage de balisage pour le formatage de documents hypertextuels --> HTML
  2. Un système de notation uniforme pour adresser les ressources accessibles sur le réseau --> Uniform Resource Identifier (URI). Pour des raisons historiques, on l'appelle le plus souvent Uniform Resource Locator (URL)
  3. Un protocole uniforme pour échanger des données entre les clients et les serveurs du Web

L'architecture du Web

Modèle TC/IP

1: Network Access Layer

L'infrastructure physique qui permet aux ordinateurs de communiquer entre eux sur Internet.

2 : Internet Layer

  • Contrôle le flux et l'acheminement du trafic afin de garantir la rapidité et la précision de l'envoi des données
  • Rassemble paquet de données à sa destination.
  • IP (Internet Protocol) et ICMP (Internet Control Message Protocol)

3: Transport Layer

  • Fournit des solutions pour des communications (connexions) fiables.
  • TCP Transmission Control Protocol
  • UDP (Datagram Protocol) pour les services - tels que les services de diffusion vidéo et audio - qui ne nécessitent pas la fiabilité (et la surcharge) associée au protocole TCP

4 : Application Layer

  • Permet à l'utilisateur d'accéder au réseau
  • Ex : courriels, applications de messagerie. services cloud
  • SMTP, FTP, SSH, HTTP, operate

Histoire du Web

Les origines d'Internet

  • 1965 : USA, Advanced Research Projects Agency (ARPA), pour résoudre les problèmes de gestion : l'ARPA possédait plusieurs ordinateurs qui avaient beaucoup de difficultés à communiquer entre eux
  • 1966 : ARPANET, sous la responsabilité de Jospeh Licklider (MIT)
  • Packet switching : les paquets de données empruntent différents itinéraires vers leur destination
  • 1969 : Les deux premiers nœuds Internet entre l'UCLA et l'Université de Standford. Application Telnet

Histoire du Web

Les origines d'Internet

  • 1973 : Robert Kahn (ARPA) et Vinton Cerf (Université de Stanford), ont formalisé par écrit la structure de Internet
  • 1973 : Le projet de protocole de contrôle de transmission (TCP) a été créé.
  • 1978 : IP, toujours au sein du projet ARPANET
  • 1986 : La "National Science Foundation" américaine a créé le réseau NSFNET pour connecter les universités américaines à leurs superordinateurs et les relier à ARPANET

Histoire du Web

Les origines d'Internet

  • 1990 : ARPANET, devenue obsolète, a été mis hors service. La NSF a été désignée pour administrer l'ensemble du réseau universitaire aux États-Unis
  • 1990 : HTTP créé à Genève par Tim Berners Lee, Robert Caillau et leur equipe

Histoire du Web

Les origines d'Internet

  • 1990 : premier site web
  • Pour penser au Web, Tim Berners Lee a été inspiré par d'autres projets tels que Xanadu et Hypercard.

Le Web

  • Tim Berners Lee contre l'idée de repenser un ensemble intertextuel
  • Un protocole très simple : HTTP pour récupérer le texte d'autres documents via des liens hypertextes.
  • Le format de texte pour HTTP a été nommé HTML, pour HyperText Mark-up Language.

Le Web

  • Syntaxe de base de HTML (paragraphes, titres, listes) fortement influencée par le SGML
  • Un protocole très simple : HTTP pour récupérer le texte d'autres documents via des liens hypertextes.
  • Clairement SGML ne comprend pas les liens
