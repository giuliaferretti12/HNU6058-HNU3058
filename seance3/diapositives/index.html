  • 1995
  • Brendan Eich
  • Netscape
  • 1997 : European Computer Manufacturers Association (ECMA)
  • Octoverse report 2018 par GitHub : there are more JavaScript code repositories than any other language
  • Frameworks : Ex. Angular, React, Vue, Svelte
  • ECMAScript Editions

Que peut-on faire avec JS ?

Manipulation des pages Web, Interaction avec l'utilisateur, et le serveur Web

Par exemple, in-browser JavaScript est capable de :

  • Ajouter du nouveau HTML à la page, changer le contenu existant, modifier les styles
  • Réagir aux actions de l'utilisateur, s'exécuter sur les clics de souris, les mouvements du pointeur, les pressions sur les touches
  • Envoyer des requêtes sur le réseau à des serveurs distants, télécharger et téléverser des fichiers (technologies dites AJAX et COMET)
  • Obtenir et définir des cookies, poser des questions au visiteur, afficher des messages
  • Mémoriser les données du côté client ("stockage local")

Lier une page HTML à un script JS

  • <script src="/path/to/script.js">
  • <script src="https://link.js">
  • Embedded style avec le tag <script>

N.B. : Plusieurs scripts peuvent être liés à un seul document HTML
